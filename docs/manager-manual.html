基本介绍

本说明书介绍【掌柜系统】的使用方法，面向KTV管理人员和运营人员。系统包含包厢管理、商品管理、员工管理、门店管理四大核心功能模块，本手册将详细说明各模块的操作方法和配置技巧。

系统登录

功能说明：系统登录是进入掌柜系统的入口，支持账号密码和手机验证码两种登录方式。

操作方法：

  1. 打开系统登录界面
  2. 输入用户名和密码，或使用手机号验证码登录
  3. 点击"登录"按钮
  4. 系统验证身份后进入主界面
系统界面

界面组成：

  • 顶部导航区：显示门店名称和账号名称
  • 中央内容区：包含包厢管理、商品管理、员工管理、门店管理四个主要功能入口
  • 底部Tab切换区：营业简报和管理Tab切换

门店列表

功能说明：门店列表用于展示当前账号可管理的所有门店，便于多门店运营的用户快速切换和管理不同门店的数据与配置。

操作方法：

  1. 进入"门店管理→门店列表"功能
  2. 页面会显示当前账号下所有已创建的门店信息，包括门店名称、地址、联系电话等
  3. 点击某一门店卡片，可切换当前操作门店，系统会自动刷新为该门店的数据和配置
  4. 如需新增门店，点击页面右上角"新增门店"按钮，填写门店基本信息
