系统简介

本说明书面向KTV服务员，介绍如何在Pad端使用移动点单系统，实现包厢现场的高效点单与服务操作。系统采用触摸屏操作，界面简洁直观，便于快速上手。

移动点单系统是专为KTV包厢现场点单设计的应用程序，通过Pad等移动设备，服务员可以在包厢内直接完成点单、查看订单、响应呼叫等操作，无需在包厢和前台之间来回奔波，显著提高服务效率和客户体验。

系统登录

功能说明：系统登录页用于验证服务员身份，保障数据安全。支持手机号+密码登录和微信扫码登录两种方式。

操作方法：

  1. 打开Pad上的移动点单系统，进入登录界面
  2. 输入手机号和密码，或选择"微信扫码登录"
  3. 可勾选"记住密码"以便下次快速登录
  4. 点击"登录"按钮，系统验证通过后进入主界面
  5. 登录界面右下角可查看门店编号、门店名称和当前设备IP地址，便于技术支持
系统登录页面截图展示（原型阶段暂不提供）

选择包厢

操作步骤：

  1. 登录后进入"实时包厢页"
  2. 可通过顶部"全部/使用中"筛选按钮，仅显示正在使用中的包厢
  3. 支持包厢搜索和筛选，快速定位目标包厢
  4. 点击目标包厢卡片，进入该包厢的点单页面
