前言

本说明书针对【汇金Pro 收银系统】的日常操作，主要面向收银员和前台服务人员。系统包含包厢管理、商品点单、订单处理、结账收款、存酒管理和交班管理六大核心模块，本说明书将详细介绍各模块的基本操作流程。

基本介绍

系统登录

  1. 打开系统登录界面
  2. 输入用户名和密码或扫描二维码
  3. 点击"登录"按钮
  4. 系统验证身份后进入主界面
  5. 不同级别账号拥有不同操作权限，部分功能可能需要更高权限账号才能执行
小贴士

首次登录系统时，建议修改默认密码以确保账户安全。密码应包含字母、数字和特殊字符，长度不少于8位。

实时包厢页面

实时包厢页是系统的主页面，显示所有包厢的实时状态和快捷操作按钮。

包厢状态颜色说明
  • 白色空闲包厢，可以开台
  • 蓝色使用中包厢
  • 黄色使用超时未结账的包厢
  • 灰色故障状态，暂不可用
包厢状态标识
  • 预订标识：包厢右上角显示"订"字，表示该包厢已被预订
  • 带客标识：包厢右上角显示"带"字，表示该包厢正在带客试用
  • 清洁标识：包厢右上角显示"扫"字，表示该包厢正在清洁中
  • 锁房标识：包厢右上角显示锁的图标，表示该包厢已被锁房，不能再点单
  • 联台标识：包厢右上角显示链接图标，表示该包厢与其他包厢联台
  • 生日标识：包厢右上角显示蛋糕图标，表示该包厢客人过生日
  • 呼叫标识：包厢右上角显示铃铛图标，表示该包厢正在呼叫服务员，有服务需求
  • 时间标识：包厢右下方显示时钟图标，表示使用中的包厢剩余的时间
  • 支付标识：包厢右下方显示支付状态，包括未结账/已结账/挂账
  • 低消标识：包厢右下方显示低消未满警示

包厢详情面板

  1. 点击任意包厢卡片，右侧会显示该包厢的详细信息
  2. 详情面板显示：
    • 包厢基本信息（包厢类型、所在区域）
    • 预订/开台时间信息（预订时间、预计结束时间）
    • 费用明细（包厢费用、商品消费、最低消费等）
    • 客人信息（姓名、电话、备注等）
  3. 根据包厢状态，面板底部显示不同的可用操作按钮

辅助功能

  1. 使用搜索框，可直接搜索包厢号或客人信息
  2. 使用排序功能，可按区域、类型、名称、消费、状态等排序包厢
  3. 使用筛选条件，可按区域、类型或状态等筛选包厢
  4. 使用网格与列表切换，可以切换实时包厢页的布局格式

1. 包厢管理

1.1 开台

开台是将空闲包厢分配给客人使用的操作。

操作步骤
  1. 在实时包厢页选择一个空闲包厢（白色）
  2. 点击"开台"按钮，进入开台页面
  3. 选择计费类型（买钟/买断/核销）
  4. 选择具体计费方案
  5. 设置开台信息：
    • 客人信息（选填）
    • 开始时间（默认当前时间）
    • 使用时长（根据计费方案自动填充）
  6. 选择结账方式：
    • 开台后结：先开台，稍后结账
    • 开台立结：立即支付费用
可选操作
  • 会员登录：点击"会员登录"按钮，输入会员卡号或手机号，系统自动应用会员价格和折扣
  • 添加商品：点击"添加商品"按钮，选择商品加入开台单
  • 商品打折：点击"商品打折"按钮，对开台配送的商品进行折扣设置
  • 自定义低消：点击"自定义低消"按钮，为未设置最低消费的包厢设置最低消费金额（需授权）
  • 计费方式调整：调整包厢的计费类型，实现价格调整，类似升舱/降舱功能（需授权）
  • 预付款：点击"预付款"按钮，客人预先支付一定金额
  • 备注：点击"备注"按钮，添加开台备注信息，可针对整单或具体商品进行备注

1.2 预订

预订用于提前为客人保留包厢。

操作步骤
  1. 在实时包厢页，选择一个包厢（通常为空闲包厢）
  2. 点击"预订"按钮，弹出预订窗口
  3. 填写预订信息：
    • 必填：选择包厢、客人姓名/电话/会员卡号、预抵时间
    • 选填：开台方案、客人来源、备注
  4. 点击"确认"按钮完成预订
  5. 预订成功后，该包厢右上角显示"订"字标识

1.3 预订开台

预订开台用于客人到店后，将已预订的包厢正式开台。

操作步骤
  1. 在实时包厢页选择一个带有"订"字标识的预订包厢
  2. 点击"预订开台"按钮，进入开台页面
  3. 系统自动填充预订信息（客人姓名、电话、预选计费方案等）
  4. 确认或修改开台信息
  5. 选择开台方式（开台后结/开台立结）
  6. 点击确认完成开台
