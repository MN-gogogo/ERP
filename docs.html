使用准备

系统环境要求和配置说明

在开始使用汇金Pro系统前，请确保您的设备满足以下基本要求，并按照配置指南进行设置，以确保系统正常运行。

  • 操作系统：Windows 10及以上版本
  • 处理器：Intel Core i3或同等性能处理器
  • 内存：4GB RAM (推荐8GB)

  • 操作系统：Android 8.0或以上
  • 设备：Android平板或大屏手机
  • 屏幕：8寸以上屏幕

  • 推荐浏览器：谷歌内核Chrome

功能概览

汇金Pro系统功能模块概述

汇金Pro系统由四大核心模块组成，每个模块承担不同的业务功能，共同构成完整的KTV经营管理解决方案。

掌柜系统

店长/老板移动管理平台

  • 实时监控 - 营业数据大盘，包厢实时状态
  • 门店管理 - 门店配置和收银机绑定
  • 包厢管理 - 包厢类型配置，价格设置
  • 商品管理 - 商品类型配置，价格调整
收银系统

KTV前台核心业务系统

  • 包厢管理 - 开台、预订、续房、换房等
  • 商品点单 - 酒水、小吃、果盘等商品销售
  • 结账收款 - 多种支付方式，优惠管理
  • 存酒管理 - 客人酒水寄存与取用
  • 交班管理 - 班次结算与交接
移动点单系统

服务员现场接单工具

  • 包厢点单 - 现场快速下单，减少等待
  • 订单跟踪 - 实时查看订单状态
  • 催单管理 - 处理客人催单请求
  • 呼叫服务 - 接收处理客人呼叫
财务报表系统

业务数据分析平台

  • 营收报表 - 日/周/月营业额统计分析
  • 商品分析 - 热销品分析，毛利统计
  • 会员分析 - 会员消费行为洞察
  • 包厢分析 - 包厢使用率与营收贡献
  • 趋势预测 - 季节性波动和趋势分析

💻 收银系统使用说明书

本说明书针对收银员和前台服务人员，涵盖日常工作中最常用的操作流程。

🚀 5个核心操作
1
开台 - 将空闲包厢分配给客人
选择白色空闲包厢 → 点击"开台" → 选择计费方案 → 确认开台
2
点单 - 为包厢添加商品消费
选择使用中包厢 → 点击"点单" → 选择商品 → 确认点单
3
结账 - 完成包厢消费支付
选择使用中包厢 → 点击"结账" → 选择支付方式 → 确认结账
4
续房 - 延长包厢使用时间
选择使用中包厢 → 点击"续房" → 选择时长 → 确认续房
5
退单 - 处理商品退款和取消
选择包厢 → 点击"退单" → 选择退单商品 → 确认退款方式
🔧 其他常用功能
预订管理 查看消费明细 并房联台 存酒管理 交班操作

KTV字典

汇金Pro系统术语词典

本字典收录了KTV行业常用专业术语及在汇金Pro系统中的具体含义，帮助新用户快速了解行业术语，提高系统使用效率。

定义：客人进入包厢并开始计费的操作。

应用场景：收银员为到店客人分配包厢并开始计时计费的过程。开台时需选择计费方式（买钟/买断）、使用时长等信息，生成消费订单。

相关术语：预订开台、带客开台

定义：结束包厢使用，停止计费的操作。

应用场景：客人使用结束后，收银员结束包厢计时计费，并清空包厢供下一位客人使用。通常在结账完成后自动执行关台操作。

相关术语：结账、退房

定义：短暂带领客人进入包厢参观的功能。

应用场景：前台接待客人时，临时标记包厢为"带客"状态，带领客人参观包厢环境。系统会暂时锁定该包厢，避免被其他客人预订或开台。

相关术语：空闲包厢、临时锁房

定义：暂时禁止对某个包厢进行操作（如点单、续房等）的功能。

应用场景：当需要控制客人消费或处理特殊情况时，管理员可对特定包厢进行锁房操作，暂停该包厢的商品点单和续时等功能，直到解除锁定。

相关术语：解锁、临时锁房

定义：客人以固定价格租用包厢一段时间的消费模式。

应用场景：客人支付固定金额后可在约定时间段内使用包厢，不再另外收取包厢费用。例如：下午场买断5小时500元。

相关术语：买钟、包厢计费

定义：按时间计费的包厢使用模式。

应用场景：根据客人实际使用包厢的时长收费，通常按小时计费。例如：小包厢100元/小时，使用3小时收费300元。

相关术语：买断、包厢计费、续房

定义：客人在使用包厢前预先支付的费用。

应用场景：开台时客人先支付一定金额作为消费押金，后续消费可抵扣该金额。预付费有助于控制客人消费额度，减少坏账风险。

相关术语：押金、开台立结

定义：将已结账的账单恢复到未结状态的操作。

应用场景：当结账操作有误或客人需要继续消费时，管理员可将已结账的订单恢复为未结状态，允许继续添加消费项目。此操作通常需要较高权限。

相关术语：反结账、账单修改
